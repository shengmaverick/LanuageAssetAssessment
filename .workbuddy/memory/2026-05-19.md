## 语言资产成熟度评估微信小程序开发完成

### 项目路径
`/Users/SJ/Downloads/test_files/LanuageAssetAssessment/miniprogram/`

### 完成内容
根据用户提供的完整问卷设计，开发了一套微信小程序：
- **3 个页面**：首页（基本信息）、问卷页（20题/5维度）、结果页（雷达图+等级）
- **数据文件**：`utils/questionData.js` 包含全部20道题、选项、分值和5个等级规则
- **HTML 预览文件**：`preview.html`，可在浏览器中交互体验完整流程

### 页面功能
1. **首页** (`pages/index`)：欢迎/维度介绍/基本信息表单（部门、职位、企业类型必填，语言对选填），填写完成后激活「开始评估」按钮
2. **问卷页** (`pages/questionnaire`)：顶部固定进度条 + 维度TAB，5个维度逐一作答，支持前后跳转，所有题目答完后才能提交
3. **结果页** (`pages/result`)：显示总分、成熟度等级（初始级/重复级/规范级/优秀级/领先级）、五维度雷达图（Canvas 2d API）、各维度进度条、改进建议输入区

### 技术亮点
- 进度实时更新，维度完成状态绿点提示
- 选项选中时显示颜色与对应维度主题色同步
- 雷达图使用 Canvas 2D API 绘制，含网格/轴线/填充区域/数据点/分值标注
- HTML 预览版完整还原小程序三页交互逻辑，含手机外框
